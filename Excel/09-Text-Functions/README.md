# Day 9 - Text Functions

## Project Overview

This project includes different text functions in Excel. The sheet stores employee details in one sheet, and I created another separate sheet to practice text functions using that data.

## Features

- Employee details table
- Prefix extraction
- Employee number extraction
- Upper case name conversion
- Lower case name conversion
- Proper case name conversion
- Name length calculation

## Functions Used

- LEFT()
- RIGHT()
- UPPER()
- LOWER()
- PROPER()
- LEN()

## Learning Outcome

Practiced different text functions by creating separate columns for each function. Learned how to extract, convert, and measure text data in Excel.

# Day 1 - Excel Student Performance Tracker

## Project Overview
This project is a basic Excel student performance tracker created for Day 1 practice. The sheet stores student marks, calculates totals and averages, determines percentages, assigns grades, and highlights performance using conditional formatting.

## Features
- Student marks table
- Total marks calculation
- Average marks calculation
- Percentage calculation
- Grade assignment using IF()
- Conditional formatting for performance highlighting
- Notes section explaining formulas used

## Functions Used
- SUM()
- AVERAGE()
- IF()

## Conditional Formatting Rules
- Green → Grade A
- Yellow → Grade B/C
- Red → Grade D

## Formula Logic

Percentage:

Student Total Marks / Total Subject Marks

Grade Logic:

- A → Percentage >= 80%
- B → Percentage >= 60%
- C → Percentage >= 40%
- D → Percentage < 40%

## Learning Outcome
Practiced Excel basics including formulas, formatting, percentage calculations, and conditional formatting.

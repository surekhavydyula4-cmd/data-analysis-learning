# Day 13 - Interactive Sales Dashboard

## Project Overview

This project is a simple interactive sales dashboard created for Day 13 practice. The dashboard uses Pivot Tables, Charts, and Slicers to analyze sales data and make the dashboard dynamic.

## Features

- Category-wise sales analysis
- Region-wise sales analysis
- Category-wise product count
- Sales comparison by category and region
- Interactive slicers for filtering
- Dashboard layout using multiple charts
- Data labels for better understanding

## Tools Used

- Pivot Tables
- Bar Chart
- Pie Chart
- Clustered Column Chart
- Slicers

## Dashboard Analysis

Category-wise Sales:

Used charts to compare total sales across product categories.

Region-wise Sales:

Used charts to compare sales performance in different regions.

Category-wise Product Count:

Used a Pie Chart to show product count percentages.

Sales by Category and Region:

Used a Clustered Column Chart to compare category sales across regions.

Slicers:

Added slicers to filter dashboard data dynamically by category, region, and product.

## Learning Outcome

Practiced creating an interactive dashboard using Pivot Tables, Charts, and Slicers. Learned how to visualize data, apply filters, and analyze sales data more effectively in Excel.

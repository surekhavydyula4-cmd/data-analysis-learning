# Day 11 - Pivot Table Practice

## Project Overview

This project is a basic Pivot Table practice sheet created for Day 11. The sheet uses sales data to summarize and analyze information in different ways using Pivot Tables.

## Features

- Category-wise total sales
- Region-wise total sales
- Product count by category
- Category and region-wise sales comparison
- Grand total calculation
- Data summarization using Pivot Tables

## Functions Used

- SUM()
- COUNT()

## Pivot Table Analysis

Category-wise Sales:

Used Pivot Table to calculate total sales for each product category.

Region-wise Sales:

Used Pivot Table to calculate total sales for each region.

Product Count:

Used Pivot Table to count total products in each category.

Category vs Region:

Used Rows and Columns together to compare category sales in different regions.

## Learning Outcome

Practiced Pivot Tables in Excel and learned how to summarize, group, count, and compare data easily. This helped me understand data analysis better.

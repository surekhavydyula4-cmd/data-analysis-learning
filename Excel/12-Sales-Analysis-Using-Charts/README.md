# Day 12 - Sales Analysis Using Charts

## Project Overview

This project is a simple sales analysis dashboard created for Day 12 practice. The dashboard uses different Excel charts to visualize sales data and make comparisons between categories and regions.

## Features

- Category-wise sales analysis
- Region-wise sales analysis
- Category-wise product count
- Sales comparison by category and region
- Dashboard layout using multiple charts
- Data labels for better visualization

## Charts Used

- Bar Chart
- Pie Chart
- Clustered Column Chart

## Chart Analysis

Category-wise Sales:

Used a Bar Chart to compare total sales across different product categories.

Region-wise Sales:

Used a Bar Chart to compare sales performance across different regions.

Category-wise Product Count:

Used a Pie Chart to show the percentage contribution of each product category.

Sales by Category and Region:

Used a Clustered Column Chart to compare category sales across different regions.

## Learning Outcome

Practiced creating different chart types in Excel and learned how to visualize data for better analysis and decision-making.

# Day 8 - Data Validation and Dropdown Lists

## Project Overview

This project is a simple student course details table created to practice Data Validation and Fee Status tracking in Excel.

## Features

- Student course details table
- Data validation for courses
- Data validation for mode
- Data validation for fee status
- Conditional formatting for fee status
- Notes section explaining the concepts used

## Conditional Formatting Rules

- Green → Paid
- Red → Pending

## Learning Outcome

Practiced Data Validation and Conditional Formatting by creating a simple student course details table. Learned how to create drop-down lists, restrict user input, and highlight fee status for better understanding.

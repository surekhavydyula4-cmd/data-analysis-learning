# Day 2 - Employee Salary Report

## Project Overview

This project is an Employee Salary Report created in Microsoft Excel to practice analytical functions, formatting, and conditional formatting.

The report contains employee details, salary information, bonus calculations, and summary statistics.

---

## Features

- Employee salary tracking
- Bonus calculation
- Total salary calculation
- Highest salary identification
- Lowest salary identification
- Employee count calculation
- Indian Rupee (₹) formatting
- Conditional formatting for salary analysis
- Professional table formatting

---

## Functions Used

### SUM()
Used to calculate Total Salary.

Example:

=Salary + Bonus

### MAX()
Used to find the highest salary.

### MIN()
Used to find the lowest salary.

### COUNT()
Used to count the total number of employees.

### COUNTA()
Can be used to count non-empty employee names.

---

## Conditional Formatting

Applied conditional formatting to highlight salary ranges:

- Green → Total Salary greater than ₹50,000
- Orange → Total Salary less than ₹30,000

---

## Skills Practiced

- Excel formulas
- Data formatting
- Currency formatting
- Conditional formatting
- Basic data analysis
- Report creation

---

## Learning Outcome

Through this project, I learned how to analyze employee salary data using Excel functions such as COUNT(), MAX(), and MIN(). I also practiced conditional formatting, currency formatting, and creating structured reports.

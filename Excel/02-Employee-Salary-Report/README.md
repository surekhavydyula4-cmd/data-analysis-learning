# Day 2 - Employee Salary Report

## Project Overview
This project is a basic Excel employee salary report created for Day 2 practice. The sheet stores employee salary details, calculates total salary, identifies highest and lowest salaries, and tracks employee count.

## Features
- Employee salary table
- Bonus calculation
- Total salary calculation
- Highest salary identification
- Lowest salary identification
- Employee count calculation
- Currency formatting
- Conditional formatting

## Functions Used
- MAX()
- MIN()
- COUNT()

## Conditional Formatting Rules
- Green → Salary > ₹50,000
- Red → Salary < ₹30,000

## Learning Outcome
Practiced salary analysis, summary calculations, currency formatting, and conditional formatting in Excel.

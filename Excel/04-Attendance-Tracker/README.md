# Day 4 - Student Attendance Tracker

## Project Overview

This project is a basic Excel student attendance tracker created for Day 4 practice. The sheet stores student attendance records, calculates attendance statistics, and highlights attendance status using conditional formatting.

## Features

- Student attendance table
- Present student count
- Absent student count
- Attendance percentage calculation
- Conditional formatting for attendance status
- Freeze Panes practice
- Find & Replace practice
- Remove Duplicates practice
- Notes section explaining concepts used

## Functions Used

- COUNTIF()

## Conditional Formatting Rules

- Green → Present
- Red → Absent

## Formula Logic

Present Count:

COUNTIF(Status Range, "Present")

Absent Count:

COUNTIF(Status Range, "Absent")

Attendance Percentage:

(Present Count / Total Students) * 100

## Learning Outcome

Practiced attendance analysis using COUNTIF(), calculated attendance percentage, applied conditional formatting, and learned Excel features such as Freeze Panes, Find & Replace, and Remove Duplicates.

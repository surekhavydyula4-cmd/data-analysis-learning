# Day 6 - Employee Bonus Eligibility Tracker

## Project Overview

For today's practice, I created an Employee Bonus Eligibility Tracker in Excel. The purpose of this sheet is to determine whether employees qualify for a bonus or promotion based on their salary and performance score.

## Features

* Employee details table
* Bonus eligibility calculation
* Promotion eligibility calculation
* Summary section
* Conditional formatting
* Formula notes

## Functions Used

* IF()
* AND()
* OR()
* COUNTIF()

## Eligibility Rules

Bonus Eligibility:

* Salary should be 50,000 or more
* Performance Score should be 80 or more

Promotion Eligibility:

* Performance Score should be 90 or more
* Or Salary should be 70,000 or more

## Conditional Formatting

* Green for Eligible / Yes
* Red for Not Eligible / No

## What I Practiced

* Using IF() to return different results based on conditions
* Using AND() to check multiple conditions together
* Using OR() when any one condition can be satisfied
* Using COUNTIF() to count eligible employees
* Applying conditional formatting to make the results easy to understand

## Learning Outcome

Through this practice, I learned how logical functions can be used to make decisions automatically in Excel. I also understood the difference between AND() and OR() and how they are used in real-world business scenarios.
